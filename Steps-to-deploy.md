# Steps to push to deployment

  - git clone

  - cd RyanAGreen.github.io

  - source ~/.nvm/nvm.sh

  - nvm install node (version 16)

  - npm install -g yarn

  - yarn

  - npm install -g gatsby-cli

  - npm install

  - git add .

  - git commit -am "changes"
  
  (I think if I make changes I only need to do the final two steps)

  - gatsby build

  - npm run deploy
